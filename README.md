# `helium`

> A GPU-accelerated math object viewer built in Rust

## Technology Stack

| Component  | Crate(s)                     |
| ---------- | ---------------------------- |
| Windowing  | `winit`                      |
| GPU        | `wgpu`                       |
| UI overlay | `egui`                       |
| Math       | `glam`, `rustfft`, `ndarray` |
